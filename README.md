## yoyaku-crm
軽量な予約管理アプリ

## Getting Started

```bash
pnpm dev
```


## 技術スタック

| Layer | Technology | Rationale |
| :--- | :--- | :--- |
| Frontend | Next.js (React), TypeScript, Tailwind |  |
| Backend | Next.js API Routes, Server Action | 単一リポジトリで開発効率を最大化 |
| ORM | Prisma | 型安全・マイグレーション容易 |
| DB | PostgreSQL | RDB |
| Auth | Auth.js | 実装が簡便 |
| Hosting | Vercel | 一旦ホスティングするにはもってこい |
